Thoth MCP governance proxy endpoint security sidecar.
Governs MCP tool calls through the Moses enforcement pipeline.

## Install

**macOS / Linux**

```bash
curl -fsSL https://install.atensecurity.com/thoth | sh
```

**Windows**
Download from Releases: https://github.com/atensecurity/thoth/releases/latest
